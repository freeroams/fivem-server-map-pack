# Tinsel Towers Lobby Fix (FiveM)

This resource fixes both Tinsel Towers Lobby for FiveM:

## Description

This resource resolves various issues with the Tinsel Towers, including interior problems, visual glitches, and collision errors.  
It is intended for **FiveM servers** and installs.

## Installation (FiveM)

1. Download or clone this repository.
2. Place the folder into your FiveM server resources directory: `resources/[maps]/tinseltowers_lobby_fix`
3. Ensure the resource is started in your `server.cfg`:`ensure tinseltowers_lobby_fix`
4. Restart your server or start the resource.

## Requirements

- FiveM server
- No additional dependencies required

## Compatibility

- FiveM only
- Compatible with most map and MLO resources
