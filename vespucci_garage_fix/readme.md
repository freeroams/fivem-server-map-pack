# Vespucci Garage Fix (FiveM)

This resource fixes both Vespucci garages for FiveM:
- Small garage
- Big garage

Both fixes are included and installed together as a single resource.

## Description

This resource resolves various issues with the Vespucci garages, including interior problems, visual glitches, and collision errors.  
It is intended for **FiveM servers** and installs both the small and big garage fixes at once.

## Installation (FiveM)

1. Download or clone this repository.
2. Place the folder into your FiveM server resources directory: `resources/[maps]/vespucci_garage_fix`
3. Ensure the resource is started in your `server.cfg`:`ensure vespucci_garage_fix`
4. Restart your server or start the resource.

## Requirements

- FiveM server
- No additional dependencies required

## Compatibility

- FiveM only
- Compatible with most map and MLO resources
- Ensure no other Vespucci garage MLOs conflict with this resource
